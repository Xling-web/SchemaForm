## 技术栈
```
vue2.0 + elementUI + scss
```

## 简介
```
因为项目开发中，特别是后台系统的开发需要定义非常多的表单组件，要是一个一个的去写的话需要花费大量的时间，
所以这个Demo就是通过配置JSON数据的形式去实现表单组件的高复用、好维护，实现用配置化方式开发表单，
完完全全就是为了高复用、可维护性，然后提升开发效率，解放生产力。
```
